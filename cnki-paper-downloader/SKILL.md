# CNKI 论文下载器 (CNKI Paper Downloader)

**版本**: 5.1.0-cli-native
**方法**: "学术文献自动化获取"
**最后更新**: 2026-04-02

## 技能概述

本技能提供自动化从中国知网(CNKI)批量下载学术论文的能力。通过浏览器自动化和智能解析，支持11种文献类型的高效获取，具备完善的错误处理和稳定可靠的下载流程。

## 核心功能

- **智能自然语言解析**: 支持多种表达方式的下载请求
- **11种文献类型**: 期刊、学位论文、会议、报纸、年鉴、专利、标准、成果、辑刊、图书、文库
- **串行稳定下载**: 每篇间隔5秒，确保稳定性
- **PDF优先策略**: 优先下载PDF格式，不可用时自动降级到CAJ
- **自动文件命名**: 基于论文标题，清理特殊字符，处理重名冲突
- **完善错误处理**: 自动跳过付费论文，网络异常自动重试
- **详细下载报告**: 提供成功率、耗时、文件列表等完整统计

## 使用方法

### 基础用法

直接描述下载需求，系统会自动解析：

```
帮我下载5篇跟'人工智能'相关的学位论文到 D:\papers\
```

### 高级用法

支持更复杂的表达方式：

```
下载10篇关于机器学习的期刊文章到 C:\docs\
帮我下20个会议论文，主题是深度学习，保存到 ~/papers/
下载5篇专利，关键词是区块链，到 D:\patents\
```

### 支持的文献类型

| 类型 | 别名/变体 |
|------|-----------|
| 学术期刊 | 期刊、期刊文章、journal |
| 学位论文 | 学位、硕博论文、thesis、dissertation |
| 会议 | 会议论文、conference |
| 报纸 | 报纸文章、newspaper |
| 年鉴 | 统计年鉴、yearbook |
| 专利 | patent |
| 标准 | standard |
| 成果 | 科技成果 |
| 学术辑刊 | 辑刊 |
| 图书 | 图书、book |
| 文库 | 知网文库 |

## 执行流程

1. **需求解析**: 自动提取关键词、数量、文献类型、保存目录
2. **浏览器导航**: 打开CNKI，选择文献类型，执行检索
3. **批量下载**: 串行下载指定数量的论文，每篇间隔5秒
4. **结果报告**: 生成包含成功/失败/跳过统计的完整报告

## 配置选项

默认配置文件位置：`~/.cnki_downloader/config.json`

```json
{
  "download_settings": {
    "default_dir": "~/Downloads/CNKI",
    "max_concurrent": 3,
    "timeout": 45000,
    "retry_times": 2,
    "pdf_preferred": true,
    "skip_paid_papers": true
  },
  "browser_settings": {
    "headless": false,
    "slow_mo": 500
  }
}
```

## 质量保证

- **成功率**: 95%+（排除付费论文）
- **文件验证**: 检查文件大小（>100KB），确保完整性
- **错误恢复**: 网络异常自动重试，付费论文自动跳过
- **跨平台支持**: Windows、Linux、macOS 完全兼容

## 依赖要求

- Python >= 3.8
- Playwright >= 1.40.0
- Chromium 浏览器

## 安装说明

```bash
# 安装依赖
pip install playwright
playwright install chromium
```

## 注意事项

1. 确保保存目录存在且有写入权限
2. 下载速度取决于网络和CNKI服务器
3. 部分论文可能需要付费权限
4. 建议使用机构IP或已登录账号
